Astrolabe User Manual

A comprehensive guide to mapping your Crystal Spheres.

Getting Started

Welcome to Astrolabe! This application allows you to create detailed, interactive maps of fantasy star systems.

Basic Controls

  • File Menu: Use the top menu to create new systems, open existing ones, and save your work.
  • System Editor: The left panel allows you to add, edit, and remove celestial bodies.
  • Drag & Drop: You can drag celestial bodies in the system editor to reorder them or place them inside groups.
